This is the program xmap which provides street maps for various cities around the country. To use this program you need to get a copy of xmap, and a data file for one or more cities. The currently available data sets are:

  • Washington DC Area (DC and parts of MD)
  • Baltimore, MD
  • Dane County, WI

    • If you are in the Computer Science Department at the University of Maryland, the program can be found in /fs/ftp/users/hollings/bin/xmap Further Information:
