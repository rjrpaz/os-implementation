Program #1 Data Files

Here are the data files for the program. The first set is the "fixed" ones that ensure that things tend to stay on the board for most of the game. The second set has data that runs off the board into negative coordinates.

  • final.data
  • life.data.1
  • life.data.2.fixed
  • life.data.2
  • final.data

    • For the final run, use a board of at least 5,000 in the x dimension and 1,000 in the y dimension. You should run it for 5,000 generations.
