CS 420 - Tools

This page describes the programs that we will be using for programming assignments.  All of these tools are free, so you are encouraged to download them and install them on your home machine.  All of them are available for both Windows and Linux (with the exception of Cygwin, which is only needed on Windows).

If you are using Linux, just make sure that gcc, make, perl, and nasm are installed, along with whatever text editor you prefer.

Cygwin

Cygwin is a collection of Unix programs that have been ported to Windows.  We will use various Cygwin programs to compile GeekOS, including

  • The gcc C compiler
  • GNU Make (for compiling and linking the kernel)
  • The bash shell (for running interactive commands)
  • Perl (required by the GeekOS makefile)
  • The nasm assembler

When compiling and running GeekOS, you will be using the cygwin bash shell to run commands (such as the make utility).

Bochs

Bochs is a complete hardware-level emulator for an Intel-based PC.  We will run the GeekOS kernel within Bochs, rather than booting it directly on a real (physical) computer.

You can download Bochs from

http://bochs.sourceforge.net/

On the computers in Grantley 123, Bochs can be invoked by running

c:\Program Files\Bochs-2.3pre3\bochs.exe

From the cygwin bash shell, the command is

/cygdrive/c/Program\ Files/Bochs-2.3pre3/bochs.exe

Note that a backslash ("\") character is needed just before the space character in "Program Files".

Generally, you will want to run bochs from within the cygwin shell, and the current directory should be the "build" subdirectory of the GeekOS project.
