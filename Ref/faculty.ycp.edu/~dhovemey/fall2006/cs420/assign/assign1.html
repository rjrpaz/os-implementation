In this assignment you will read the source code for a stripped-down version of GeekOS to understand how the code is organized and what functions are available.

Getting Started

From the course website in My YCP, download project0.zip from the lecture1 subdirectory in the Files section.  Unzip anywhere that is convenient.

A GeekOS project directory has the following subdirectories:

Subdirectory
 Purpose
build
 This directory contains the Makefile and is where object/executable files are generated.
include
 Contains all of the header (.h) files.
scripts
 Contains shell scripts and perl scripts needed to compile GeekOS.
src
 Contains all of the source (.c) files.

The include and src directories are further divided into subdirectories.  The primary header and source files you will need to look at for this assignment are in the include/geekos and src/geekos directories.  These are the files that are used to build the GeekOS kernel.

Your Task

Answer the following questions.

  1. What file(s) contains the code for the GeekOS boot sector?  Briefly describe what it does.
  2. What file(s) contains the code for the GeekOS second stage loader?  Briefly describe what it does.
  3. What file(s) contains the Main function?  Briefly describe what it does.
  4. What function is responsible for making an inventory of system memory?
  5. What file(s) implement memory allocation?  What functions are available for memory allocation?  What do they do?
  6. What file(s) contain support for enabling and disabling interrupts?  What functions are available for this purpose?
  7. What file(s) are involved in setting up interrupt handler functions?  What functions are available for this purpose, and what do they do?
